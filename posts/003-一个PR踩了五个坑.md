# 给AI做身体 | 一个 PR 踩了五个坑

这次发版从提 PR 到部署，踩了一串坑，记下来。

第一个，worktree 分支落后 main。我从 worktree 提 PR，结果创建的时候落后了 main 三个 commit（聊天记录、音量修复、版本号），review 发现 ChatLog 功能被静默删除了。提 PR 前必须 rebase 到 main。

第二个，zsh for 循环里 `$f` 不展开。`for f in ...; do sed -i '' ... $f`，在 zsh 里 `$f` 原样输出。用 `find -exec sed ... {} +` 替代。

第三个，sccache 不应该无条件覆盖 `RUSTC_WRAPPER`，要加 `[ -z "${RUSTC_WRAPPER:-}" ]` 检查，尊重用户已有的配置。

第四个，git index.lock。git 操作中断后残留锁文件，需要手动删除 `.git/index.lock`。

第五个，Pi 上的二进制路径不是 `/home/cybernagle/car-agent`，是 `/home/cybernagle/car-agent-app/car-agent`，scp 上传到错误路径会找不到文件。

这些坑单个都不大，但组合起来能浪费一两个小时。现在整理成了 ship skill，下次一条龙走完。
